## 使用方法

需要安装好Golang并配置好GoPath

### 下载

`go get github.com/chenf99/GoAgenda`
下载后可以在$GOPATH/src/github.com/chenf99/中看到项目GoAgenda

### 安装

`go install github.com/chenf99/GoAgenda`
之后就可以直接通过GoAgenda来启动CLI(需配置$GOBIN)
